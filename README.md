# Easy Webpack

An easy bundler referring to Webpack.

## Start

```bash
# clone with Git Bash
git clone https://github.com/FishPlusOrange/easy-webpack.git

# change directory
cd easy-webpack

# install dependencies
yarn install

# build for bundling
yarn run build
```
